# User Guide - Opportunities

## Purpose

Opportunities track qualified sales work. They link accounts, contacts, decision makers, service offerings, values, stage progress, and win/loss outcomes.

Known process flow: 1. Qualify → 2. Discovery → 3. Proposal → 4. Negotiate → 5. Close (Won or Lost).

Known statuses: New, Engaged, Delayed, Disengaged, Inactive.

## Create an Opportunity

1. Open Pipeline > Opportunities.
2. Select New.
3. Enter opportunity name, account, contact, decision maker, source, priority, and estimate.
4. Save.

## Close (Stage 5)

The final BPF stage is **Close**. At this stage, mark the opportunity outcome as Won or Lost.

For won opportunities, record actual revenue. For lost opportunities, record lost reason. Completing this stage triggers the `Opportunity - Close` Power Automate flow, which handles the outcome automation.
