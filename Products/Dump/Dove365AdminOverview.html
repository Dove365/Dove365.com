Dove365 Admin overview

Manage your Dove365 environment

The Dove365 Admin app is the central administration hub for your Power Platform environment. Use it to manage users, assign security roles, configure email, set up SharePoint document integration, and maintain the service catalogue.

Requires: Dove365 Global Admin role

Admin first steps

  1. 1Assign the Dove365 Global Admin role to your administrator account before anything else.
  2. 2Add users, assign Dove365 security roles, and create teams and positions.
  3. 3Enable and test user mailboxes for email tracking and server-side synchronisation.
  4. 4Create Service Offerings and confirm SharePoint document integration is working.
Purpose of the app

One place for environment administration

The Dove365 Admin app brings together user management, mailbox configuration, SharePoint document settings, and data configuration into a single focused app. Admins with the Dove365 Global Admin role use it to set up and maintain the environment so CRM users can work effectively.

Admin paths

Common starting points

New environment setup

Follow this sequence when setting up a fresh Dove365 deployment.

  1. Confirm Dove365 Common and CRM Starter are imported.
  2. Assign Dove365 Global Admin role to the admin account.
  3. Add CRM users and assign Dove365 CRM User or CRM Admin roles.
  4. Create teams where relevant and assign roles at team level.
  5. Set up positions for reporting hierarchy.
  6. Approve and test mailboxes for server-side sync.
  7. Confirm SharePoint document locations are configured.
  8. Create the standard Service Offerings catalogue.
  9. Optionally deploy Dynamics 365 App for Outlook to users.

Ongoing admin tasks

Day-to-day tasks to keep the environment healthy.

  1. Add or deactivate users as the team changes.
  2. Adjust role assignments using the Security Role Manager.
  3. Review mailbox sync status and resolve errors.
  4. Update or add Service Offerings as the business evolves.
  5. Check Power Automate flow health and connection references.
  6. Review SharePoint document locations if integration issues arise.
  7. Assign or remove Dynamics 365 for Outlook roles as needed.
Key admin reminders

What every Dove365 admin should know

Global Admin role is the gateway

The Dove365 Global Admin role is required to access the Admin app. Assign it before attempting to use any admin functions, and protect it carefully.

Security Role Manager is the role tool

Use the Security Role Manager control on User and Team forms to assign and remove Dove365 roles. Changes take effect immediately without a form save.

Mailboxes need approval and testing

Approving a mailbox email address and then running Test & Enable are both required steps. Skipping either one will prevent email tracking from working.
