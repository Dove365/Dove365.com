Documentation / Release Notes

Release Notes

What changed, what’s included, and what to verify after deployment. Covers the initial release of Dove365 CRM — Starter.

Dove365 CRM - Starter  ·  Version 1.1.0  ·  Initial Release

Initial Dove365 CRM — Starter release for small to medium-sized businesses needing a lightweight CRM built on Microsoft Power Platform and Dataverse. Solution versions: Dove365 Common 1.1.0.0  ·  Dove365 CRM — Starter 1.1.0.0

+

New Features

  • Model-driven app Dove365 CRM — Starter with navigation for Dashboard, Relationships, Pipeline, and Service.
  • Account and Contact forms and views — the core customer relationship record.
  • Lead table, lead process, and conversion and deactivation workflows.
  • Opportunity process with service offering relationship and pipeline management.
  • Case process with escalation and resolution fields for structured support handling.
  • Service Offering catalogue for consistent case and opportunity categorisation.
  • Dove365 CRM User and Dove365 CRM Admin security roles for role-based access control.
  • Flow failure notification child flow included in Dove365 Common.
  • Outlook service account connection reference (dove365_OutlookConnectServiceAccount).
  • Dataverse service account connection reference (dove365_DataverseConnectionServiceAccount).
  • Power Automate flow Opportunity — Close, replacing classic on-exit Opportunity Won/Lost workflows.
  • Power Automate flow Case — Resolve, replacing classic on-exit Case Resolve workflow.
  • Power Automate flow Lead — Convert, replacing classic on-exit Lead Convert workflow.
  • LinkedIn Badge PCF control and Dove365 theme with app icons.
↓

Deployment Notes

Install Dove365 Common first, then install Dove365 CRM — Starter. The Common solution provides shared infrastructure that Dove365 CRM — Starter depends on — including the flow failure notification flow and shared connection references.

For full deployment prerequisites, environment requirements, and step-by-step setup instructions, see the Environment Setup guide.

!

Known Limitations

  • The generative page may require further work or removal if dependent configuration files cause deployment issues in certain environment configurations.
  • AI features are not included in the base release. AI extensions are available as a separate engagement. View the product page.
  • Service Offering pricing fields are not a full product catalogue, quote, order, or invoicing module. They are reference fields for categorisation only.
  • SLA and entitlement automation is not included in the base support module.
✓

Post-Deployment Checks

Verify the following after each deployment to confirm the system is correctly configured and ready for users.

App opens without errors
Security roles exist and are assigned
Business Process Flows render on records
Environment variables are populated
Outlook connection reference is connected
Dataverse Intelligence is enabled
Dataverse Model Context Protocol is enabled
SharePoint document management works for Contacts
Mailboxes and email tracking work for pilot users
Questions about this release or the deployment process? Contact Dove365 — we handle every deployment directly and will walk you through any issues.
