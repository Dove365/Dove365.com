Step-by-Step Processes

Use the step list component for sequential procedures where order matters. Each step has a numbered bubble, a bold action label, and supporting detail.

  1. 1
    Open the target environment

    Navigate to make.powerapps.com and select the correct Dataverse environment from the environment picker in the top right.

  2. 2
    Import the solution

    Go to Solutions → Import solution and upload the managed solution zip file. Accept defaults and select Import.

  3. 3
    Configure connection references

    After import, open Connection References from the solution components and sign in both references with the dedicated service account.

  4. 4
    Validate

    Open the model-driven app and confirm the sitemap, BPFs, and flows are active and the theme renders correctly.
